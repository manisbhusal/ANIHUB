# ANIHUB Master Data Engine

A high-performance Anime Data API built with Flask. This engine merges data from AniList and TMDB to provide a comprehensive suite of anime information.

## 🚀 Features
* **Trending & Popular**: Real-time trending and all-time popular lists.
* **Seasonal**: Data for the current anime season (Spring 2026).
* **Upcoming & Recent**: Most anticipated releases and recently updated metadata.
* **Airing Schedule**: 7-day forecast for upcoming episodes.
* **Deep Metadata**: Character lists, Voice Actors (Seiyuu), and Studio info.
* **Search**: Advanced search functionality across the entire database.

## 🛠️ Setup
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file and add your `TMDB_API_KEY`.
4. Run the engine: `python app.py`
